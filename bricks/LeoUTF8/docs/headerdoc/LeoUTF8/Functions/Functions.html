LeoUTF8CaseFold

Applies Unicode case folding to a NUL-terminated UTF-8 C string. 

LeoUTF8Status LeoUTF8CaseFold(
    const unsigned char *text, 
    unsigned char **out, 
    size_t *outLen);
Parameters
text
Pointer to a NUL-terminated UTF-8 C string.

out
On success, receives a newly allocated UTF-8 buffer. This pointer must not be NULL.

outLen
On success, receives the byte length of the returned buffer excluding the terminating NUL. This pointer must not be NULL.

Return Value

LEO_UTF8_OK on success, otherwise a LeoUTF8Status error value.

Discussion

Convenience wrapper around LeoUTF8CaseFoldBytes().

This function treats text as a NUL-terminated C string and therefore cannot be used to case-fold buffers containing embedded NUL bytes.

This is a Unicode case-folding operation. It is not HFS+ filename comparison policy and must not be used as a substitute for measured HFS+ filename comparison behavior.

On success, out receives a newly allocated NUL-terminated UTF-8 byte buffer and outLen receives its byte length excluding the terminating NUL.

The caller owns the returned buffer and must release it with LeoUTF8Free().

LeoUTF8Free

Releases a buffer allocated by LeoUTF8Core. 

void LeoUTF8Free(
    unsigned char *text);
Parameters
text
Buffer returned by a LeoUTF8Core allocation function, or NULL.
Discussion

Releases a buffer returned by LeoUTF8Core allocation functions such as LeoUTF8NormalizeNFCBytes(), LeoUTF8NormalizeNFDBytes(), LeoUTF8CaseFoldBytes(), and their C-string convenience wrappers.

Passing NULL is allowed.

Do not use free() directly on LeoUTF8-allocated buffers. Use LeoUTF8Free() so ownership stays within the LeoUTF8Core contract.

LeoUTF8ValidateBytes

Validates an explicit UTF-8 byte buffer. 

LeoUTF8Status LeoUTF8ValidateBytes(
    const unsigned char *bytes, 
    size_t length);
Parameters
bytes
Pointer to the UTF-8 byte buffer.

length
Number of bytes to validate.

Return Value

LEO_UTF8_OK if the byte buffer is valid UTF-8, otherwise a LeoUTF8Status error value.

Discussion

Validates exactly length bytes.

This function does not assume NUL-terminated input and does not normalize the input buffer.

Embedded NUL bytes are allowed when the surrounding UTF-8 byte sequence is otherwise valid.

LeoUTF8Version

Returns the underlying UTF-8 engine version. 

const char *LeoUTF8Version(
    void);
Return Value

A pointer to a static NUL-terminated engine version string.

Discussion

Returns the version of the underlying UTF-8 processing engine used by LeoUTF8Core.

This is not the LeoUTF8 brick version. Use LeoUTF8BrickVersionString() for the LeoUTF8 project-level brick version.

The returned string is static and must not be freed.
