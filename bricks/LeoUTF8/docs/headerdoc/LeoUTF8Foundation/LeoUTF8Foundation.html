LeoUTF8Foundation

Foundation bridge for LeoUTF8Core.
Includes: "LeoUTF8.h"


LeoUTF8Foundation provides the Foundation bridge for the LeoUTF8 integration brick.

It converts explicit UTF-8 byte buffers to NSString objects and NSString values back to UTF-8 NSData values.

This layer follows Mac OS X 10.5.8 Leopard manual retain/release ownership rules.

Returned Objective-C objects are owned by the caller and must be released by the caller. No ARC assumptions are made.
